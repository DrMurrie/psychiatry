Psychiatric Tool by Dr Benjamin Murrie

[Amazing ADHD assessment generator](adhd2.html) This has everything you need. An interview template to follow. Titration algorithm. Titration and follow up plan generator. Creates a nice adhd report, certificate, script text for e scripts, Has prescriber guidance per state. Everything. You can leave everything and Autodiagnose if you want and generate a lovely report. Perfect for beginners and experienced psychiatrists alike. 

[ADHD Handout Generator](ADHD-Patient-Guide-Generator-30.html)
This is an excellent web app that you can use to provide psychoeducation to your patient, and create a titration schedule for them. It generates a nice handout to print or send to your patients. Plus generates the plain text to put in your eScript. You can choose how much info to add to the hand out. Some patients need more and some need less. It also includes non-pharm options.
